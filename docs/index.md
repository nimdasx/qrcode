# LensaQR Documentation

Welcome to the documentation for the **LensaQR** Flutter application.

## Overview
A modern Flutter application for scanning QR codes and barcodes with persistent history tracking.

## Quick Links
- [Installation Guide](../README.md#installation)
- [Features](../README.md#features)
- [Building for Release](../README.md#building-for-release)

## Core Functionality
The app uses `mobile_scanner` for high-performance scanning and `Hive` for lightweight, fast local storage of scan history.

For more detailed information, please refer to the main [README](../README.md).
